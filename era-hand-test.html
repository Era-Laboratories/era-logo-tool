Text Content

This is a regular paragraph of text. The Era Hand widget replaces the default system cursor with an SVG hand rendered from captured finger poses. Each pose defines five fingers (thumb, index, middle, ring, pinky) with parameters like rectBezOrCircle (0 = rectangle/extended, 1 = circle/curled), position offsets, size, rotation, and color. The widget listens for mousemove events and swaps poses based on CSS-selector hover rules.

How It Works

  • Default pose: open -- all fingers extended (rbc near 0), spread out from center
  • Over links/buttons: pointer -- index finger extended, others curled
  • Over .draggable: grab -- all fingers curled inward

Hover rules are set with hand.on('a, button', 'pointer') and hand.on('.draggable', 'grab'). The widget matches the element under the cursor against registered selectors and picks the first matching pose.

Developer note: The pose data below uses hardcoded placeholder finger positions. Capture real poses in the logo tool and paste the JSON here to replace the placeholder data.
Touch devices: the hand cursor is hidden on touch input. The widget listens for touchstart to disable itself and falls back to native touch behavior.
